Welcome to Insecurity of Things

Creating POC for IoT devices to promote increased IoT security. Often it is necessary to demonstrate risk in order to create action. Unfortunately, these kinds of issues don't show up on Nessus scans, so we write exploits. Please use this code responsibly.
